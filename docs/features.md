# ฟีเจอร์แต่ละเฟส

---

## เฟส 1 — ส่งมหาวิทยาลัย

**ฟีเจอร์หลัก**
- ดูข้อมูลคอนเทนเนอร์ — เรียกดูประเภทคอนเทนเนอร์และขนาดที่มีอยู่
- ดูรายการสินค้า — เรียกดูข้อมูลสินค้าที่กำหนดไว้
- หน้าตั้งค่า — ดูข้อมูลคอนเทนเนอร์และสินค้า (อ่านอย่างเดียว)

**หน้าวางแผนบรรจุคอนเทนเนอร์ (หน้าแยก)**
- เลือกประเภทคอนเทนเนอร์
- เพิ่มสินค้าพร้อมจำนวนกล่องที่ต้องการบรรจุ
- กำหนดรูปแบบการวางกล่องในแต่ละชั้นด้วยตนเอง (Interlock) — แต่ละสินค้ามีรูปแบบชั้นวางที่แตกต่างกัน
  - รูปแบบ Interlock คือเทคนิคการวางลังสินค้าในแต่ละชั้นให้มีรูปแบบแตกต่างจากชั้นที่อยู่ติดกัน
    โดยการสลับตำแหน่ง (เช่น ซ้าย-ขวา, ข้างใน-ข้างนอก) หรือเปลี่ยนทิศทางกล่อง (แนวตรง/แนวขวาง)
    เพื่อไม่ให้รอยต่อของลังตรงกันในแนวตั้ง — ทำให้กองสินค้ายึดเกาะกันและมีความเสถียรสูง
  - ตรงข้ามกับการเรียงแบบ "ธรรมดา" ที่วางลังทับตำแหน่งเดิมซ้ำทุกชั้น (เร็วกว่าแต่เสถียรน้อยกว่า)
  - แต่ละสินค้ามีรูปแบบ Interlock เฉพาะของตัวเอง ซึ่งผู้ใช้กำหนดเองต่อชั้น
  - _(UI สำหรับกำหนดรูปแบบชั้น — ยังไม่ได้ออกแบบ)_
- แสดงผลภาพ Isometric 2.5D — มุมมองกล่องในคอนเทนเนอร์แบบ perspective
  - แสดงกล่องที่บรรจุได้จริง แยกสีตามประเภทสินค้า
  - แสดงจำนวนกล่อง และพื้นที่ที่ใช้ไป

**ระบบลิขสิทธิ์และการควบคุมสิทธิ์**
- กรอกรหัสลิขสิทธิ์ตอนเปิดโปรแกรมครั้งแรก (ต้องต่ออินเทอร์เน็ต)
- ผูกลิขสิทธิ์กับเครื่อง (HWID) — ใช้ได้เครื่องเดียว
- โหมดอ่านอย่างเดียว หากออฟไลน์หรือเครื่องไม่ตรง (UI แสดงปกติ แต่แก้ไขไม่ได้)
- UI ภาษาไทยทั้งหมด

**ข้อจำกัด**
- ข้อมูลคอนเทนเนอร์และสินค้าถูกกำหนดโดยนักพัฒนาก่อนส่งมอบ — ผู้ใช้ไม่สามารถเพิ่ม/แก้ไข/ลบได้
- ต้องต่ออินเทอร์เน็ตทุกครั้งที่เปิดโปรแกรมเพื่อตรวจสอบลิขสิทธิ์

## เฟส 2 — โรงงาน (แบบชำระเงิน)

**ทุกอย่างในเฟส 1 บวกเพิ่ม:**

**จัดการการตั้งค่า**
- เพิ่ม / แก้ไข / ลบ ประเภทคอนเทนเนอร์
- เพิ่ม / แก้ไข / ลบ สินค้า
- นำเข้า / ส่งออก ข้อมูลคอนเทนเนอร์และสินค้า (JSON)
- จัดการเทมเพลตสินค้า

**ฟีเจอร์การใช้งาน**
- _(ยังไม่กำหนด — เช่น การวางแผนการบรรจุคอนเทนเนอร์, รายงาน ฯลฯ)_

**การติดตั้ง**
- ไม่มีระบบลิขสิทธิ์ — ส่งมอบซอร์สโค้ด ลบไฟล์ auth ออกเอง
- ผู้ใช้ควบคุมการตั้งค่าทั้งหมดได้เต็มที่
- ทดลองใช้ฟรี 30 วัน ก่อนชำระค่าลิขสิทธิ์
